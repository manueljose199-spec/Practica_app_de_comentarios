## 📖 Descripción

Este proyecto es una aplicación web básica que permite a los usuarios **escribir, visualizar y eliminar comentarios** relacionados con libros.  
Está diseñada para practicar los fundamentos de **JavaScript**, la manipulación del **DOM** y el manejo de eventos en formularios.

La aplicación funciona completamente del lado del cliente y no requiere base de datos ni backend.

---

## 🛠️ Tecnologías Utilizadas

- **HTML5** – Estructura semántica del sitio  
- **CSS3** – Estilos y diseño visual  
- **JavaScript** – Lógica, eventos y manipulación del DOM  

---

## 🎯 Objetivo del Proyecto

## ⚙️ Funcionamiento

1. El usuario escribe un comentario en el área de texto.
2. Al presionar **“Agregar comentario”**:
   - Se evita la recarga de la página.
   - Se valida que el comentario no esté vacío.
   - Se crea un nuevo comentario dinámicamente.
   - Se muestra la fecha y hora de publicación.
3. Cada comentario incluye un botón **Eliminar** que permite borrarlo individualmente.

---

## 🖥️ Características Principales

- ✅ Agregar comentarios dinámicamente  
- ✅ Mostrar fecha y hora del comentario  
- ✅ Eliminar comentarios individualmente
